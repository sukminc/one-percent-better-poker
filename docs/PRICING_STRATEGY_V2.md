# onepercentbetter.poker Pricing Strategy V2

## 1. Strategic Positioning

onepercentbetter.poker should not compete head-on with solver-first products like GTO Wizard.

Their value proposition is:

- solution access
- theory study
- hand analysis against solver baselines
- high-volume training

Our value proposition is different:

- upload real hand histories fast
- surface fish-field context
- summarize exploit opportunities
- remember player habits over time
- turn raw files into personal pattern intelligence

In short:

`GTO tools teach theory. We reveal how you actually attack soft fields.`

That means pricing should be:

- meaningfully cheaper than solver platforms
- easier to understand
- structured around workflow and memory, not solver depth

## 2. Market Reference

As a reference point, third-party pricing summaries for GTO Wizard's MTT plans show annualized monthly prices roughly starting around:

- Starter: about `$35-$39/month`
- Premium: about `$62-$69/month`
- Elite: about `$116-$129/month`

Sources:

- [PokerNews plan comparison](https://www.pokernews.com/poker-tools/gto-wizard/plans-and-pricing.htm)
- [PokerNews review pricing table](https://www.pokernews.com/news/promotions/2026/01/gto-wizard-pokerarena-season-7-50347.htm)
- [PokerCorner pricing summary](https://pokercorner.io/tools/gto-wizard)

We should stay clearly below this range because:

- we are not delivering solver libraries
- we are not selling GTO study depth
- we are selling review workflow and exploit insight

## 3. Proposed Tier Structure

Launch with three tiers:

- `Lite`
- `Pro`
- `Hook`

This creates a strong funnel:

- Lite reduces friction
- Pro becomes the core paid product
- Hook is the memory-and-coaching tier for serious repeat users

## 4. Tier Definitions

## Lite

### Role

`The easiest possible first experience.`

Designed for mobile-first intake and low-friction discovery.

### Entry Experience

- user downloads hand history to iPhone or cloud drive
- user uploads from mobile
- one very lightweight report is generated
- no heavy dashboard
- no detailed drill-down

### Included Features

- mobile upload
- drag-and-drop or file-picker style upload from iPhone/iCloud/Dropbox/Files
- one file or one report at a time
- very simple tournament summary
- one short exploit-oriented summary sentence unavailable in GTO-style products

Example summary types:

- "You handled soft-passive pressure well, but likely missed some thin value."
- "You played disciplined baseline poker, but did not punish limp-heavy spots enough."
- "Against weak fields, your line looked solid but too honest."

### Limits

- no persistent history
- no cross-tournament memory
- no pattern tracking
- no heatmaps
- no weekday/hour trends
- no average bullets
- no ITM trends
- no desktop workspace

### Purpose

Lite is the acquisition engine.
It proves the core insight language and gets users into the ecosystem quickly.

## Pro

### Role

`The main desktop analysis workspace.`

This is where the real product starts.

### Entry Experience

- user uploads from desktop
- can review richer single-tournament reports
- can access web account
- can revisit stored reports

### Included Features

- desktop access
- web login and report history
- richer per-tournament review
- multi-file upload
- tournament summaries
- more detailed exploit summary
- field texture indicators
- fish-response sections
- blurred or limited longitudinal previews

### Pro Focus

Pro is still mostly about:

- reviewing tournaments
- understanding field texture
- seeing deeper exploit summaries

But not yet the full "memory engine."

## Hook

### Role

`The full habit and exploit identity system.`

This is the premium retention tier.

### Included Features

- everything in Pro
- persistent long-term memory
- calendar heatmap
- weekday volume pattern
- hourly volume pattern
- session rhythm analysis
- average bullets per tournament
- ITM rate and trend
- repeated fish-response tendencies
- decision pattern memory
- cumulative exploit identity
- folder auto-sync
- future global-pool memory and personalization

### Why "Hook"

This tier should feel like:

- the system knows you
- the system remembers your habits
- the product becomes hard to leave

This is the strongest subscription moat.

## 5. Platform Mapping

The tiers should map to platform experience clearly.

### Mobile

- Lite is ideal on mobile
- upload from Files / iCloud / Dropbox
- get one clean summary
- upgrade prompt leads to account creation

### Desktop

- Pro and Hook should shine on desktop
- this is the natural home for deeper reports and file workflows

### Web

Web should not try to replace desktop upload at first.
It should support:

- login
- billing
- report history
- account management
- teaser dashboards
- upgrade prompts

Later, web can become a true report surface for paid users.

## 6. Recommended Pricing

These are recommended launch prices, intentionally below solver-first products.

### Lite

- `Free`

Rationale:

- easiest entry
- best viral/discovery path
- builds trust fast

### Pro

Recommended:

- `$12/month` billed monthly
- `$96/year` billed annually
- effective annual price: `$8/month`

Alternative higher test:

- `$15/month`
- `$120/year`

### Hook

Recommended:

- `$24/month` billed monthly
- `$192/year` billed annually
- effective annual price: `$16/month`

Alternative higher test:

- `$29/month`
- `$228/year`

## 7. Why This Pricing Works

This keeps the product clearly cheaper than solver subscriptions while still communicating value.

It also aligns with willingness to pay:

- users will not compare us to "another database tool"
- they will compare us to GTO Wizard, Poker Coaching, and other poker study products
- being materially cheaper helps position us as a practical exploit companion, not a theoretical luxury

## 8. Annual Plan Strategy

Annual should be the default push.

Recommended annual discount:

- roughly `30-35%` off monthly effective price

Messaging examples:

- "Build your exploit profile over a full year"
- "Soft-field edges compound over time"
- "Annual saves 4 months"

Annual matters more for us than for solver products because our value grows with remembered behavior.

## 9. Upgrade Logic

### Lite -> Pro

Trigger when user wants:

- desktop review
- full tournament report
- report history
- deeper exploit breakdown

### Pro -> Hook

Trigger when user wants:

- memory
- pattern tracking
- heatmaps
- average bullets
- ITM trends
- recurring habit analysis

This is important:

`Pro sells report depth. Hook sells remembered identity.`

## 11. Web Product Role

The web product should support three jobs:

- acquisition
- conversion
- continuity

At launch, web should let users:

- learn the product
- create account
- subscribe
- view saved reports if paid
- manage billing

Over time, web can expand into:

- trend dashboard
- lightweight review surface
- cross-device continuation

## 13. Recommendation

The cleanest launch model is:

- `Lite` for mobile-first discovery
- `Pro` for desktop tournament review
- `Hook` for long-term memory and retention

This structure reflects the real product better than copying solver-tier pricing logic.

We are not charging for more theoretical power.
We are charging for:

- easier intake
- sharper exploit interpretation
- remembered user behavior
- cumulative edge discovery
